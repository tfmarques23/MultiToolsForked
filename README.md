# 🛡️ MultiToolsForked - Cybersecurity Toolkit

Este repositório serve como um **HUB centralizado** para ferramentas de Pentest, Red Team e Blue Team. Utiliza `git submodules` para manter referências atualizadas aos repositórios originais e forks essenciais.

---

## 📂 Estrutura do Toolkit

### 🏗️ Active Directory (AD)
Ferramentas para enumeração, ataques e defesa de ambientes Windows/AD.

| Ferramenta | Descrição |
| :--- | :--- |
| **Impacket** | Coleção de classes Python para trabalhar com protocolos de rede. |
| **Responder** | LLMNR, NBT-NS e MDNS poisoner. |
| **PetitPotam** | Ferramenta para forçar autenticação Windows. |
| **BloodHound.py** | Ingestor Python para o BloodHound. |
| **GPOHound** | Ferramenta para analisar vulnerabilidades em GPOs. |

### 🌐 Web Hacking
Scripts e ferramentas para auditoria de aplicações Web.

| Ferramenta | Descrição |
| :--- | :--- |
| **PayloadsAllTheThings** | Guia gigante de payloads e bypasses para Web. |
| **Trufflehog** | Scan de segredos, chaves e tokens em repositórios. |
| **Evilginx2** | Framework de Phishing para bypass de MFA. |
| **Commix** | Ferramenta de exploração de Command Injection. |

### 🔍 Enumeração & Recon
Scripts para automação de scans e coleta de informações.

| Ferramenta | Descrição |
| :--- | :--- |
| **SecLists** | A lista definitiva de wordlists para segurança. |
| **NmapAutomator** | Script de automação para scans de Nmap rápidos. |
| **Raccoon** | Ferramenta ofensiva para reconhecimento e análise de vulnerabilidades. |

### 📜 Notas & Cheatsheets
Material de estudo e guias rápidos para certificações e operações.

| Material | Categoria |
| :--- | :--- |
| **OSCP-Stuff** | Recursos focados na certificação OSCP. |
| **Blue-Team-Notes** | Documentação focada em defesa e monitoramento. |
| **LOLBAS** | Binários do Windows que podem ser usados para propósitos ofensivos. |
